## About This Repository

This repository contains the lab reports and project work completed during the **AWS Cloud Institute** program — a hands-on, skills-based training program covering cloud development, cloud operations, and cloud fundamentals on Amazon Web Services.

Over the course of the program I built real applications, deployed cloud infrastructure, wrote Python code to interact with AWS AI services, managed Linux servers, and implemented security monitoring — all within a live AWS environment.

---

## What I Learned

###  Cloud Development
Built Python applications that integrate with AWS AI and cloud services using Boto3. Worked with **Amazon Bedrock** to generate AI-powered text using foundation models like Amazon Nova and Titan, experimented with inference parameters like temperature to shape model output, and chained multiple services together into end-to-end AI pipelines.

###  Artificial Intelligence & Machine Learning Services
Applied several AWS AI services in practical applications:
- **Amazon Bedrock** — text generation with Nova Lite and Nova Micro foundation models
- **Amazon Rekognition** — object and label detection from images
- **Amazon Translate** — real-time multi-language content translation
- **Amazon Polly** — text-to-speech audio generation
- **Amazon Textract** — intelligent text extraction from documents

###  Cloud Operations & Linux Administration
Developed hands-on Linux system administration skills including file system management, user and group administration, file permissions, and package management with `dnf`. Deployed and managed services using `systemctl`, monitored system performance with `top`, and used AWS Session Manager for secure, keyless server access.

###  Infrastructure as Code
Authored and deployed **AWS CloudFormation** templates with parameterization, resource dependencies, and dynamic outputs. Detected and remediated configuration drift, created change sets for safe stack updates, and automated drift remediation using **AWS Lambda** and **Amazon EventBridge**.

###  Security & Monitoring
Implemented security monitoring using **Amazon CloudWatch** — installing the CloudWatch agent, ingesting authentication logs, creating metric filters for failure detection, and configuring SNS-based alarm notifications. Monitored NAT gateway traffic for data exfiltration signals and reviewed workload architecture using the **AWS Well-Architected Tool**.

###  DevOps & CI/CD Foundations
Used **AWS CodeCommit** for Git-based source control, **AWS CodeDeploy** for automated application deployments to EC2 fleets, and **AWS Systems Manager Parameter Store** for centralized configuration management across CloudFormation stacks.
